    - JWT : 완벽하게 Stateless, 다만 데이터가 많아지면 네트워크 트래픽 증가, 클라이언트에서 처리 성능에 영향,
    - Redis : TTL로 만료 관리해서 Stateless, 변경 일단 가능함 POST니까,
